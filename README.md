# Meu Portfólio Pessoal 👾

Este projeto é uma apresentação pessoal do meu trabalho, habilidades e projetos. Ele foi construído com foco em uma experiência interativa e visualmente atraente.

![Portfólio](https://github.com/FelipeFidalgo19/Portifolio/assets/49208754/234ffa00-95f1-4863-ba32-e8159fad4838)

## 🎯 Objetivos do Projeto (Plano de Desenvolvimento)

- **Estrutura ágil:** Ambiente de fácil inicialização, preferencialmente utilizando um script (ex: arquivo `.bat`) para rodar o projeto rapidamente.
- **Frontend Interativo:** Interface capaz de enviar informações de contato e monitorar interações (como novos acessos de usuários).
- **Sistema de Identificação e Analytics:**
  - Implementação de cookies/registro para identificar usuários novos, cadastrados e anônimos.
  - Rastreamento e notificação de quando um usuário realiza o login ou acessa a página, ajudando na geração de métricas do sistema.

## 🛠 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes ferramentas principais (com base no `package.json`):

- **[React](https://reactjs.org/)** (v18)
- **[Material UI](https://mui.com/)** (@mui/material, @mui/icons-material)
- **[Firebase](https://firebase.google.com/)** (Autenticação / Banco de dados)
- **[Three.js](https://threejs.org/)** (Gráficos e visualizações 3D)
- **[Framer Motion](https://www.framer.com/motion/)** (Animações fluidas)
- **[Sass](https://sass-lang.com/)** (Estilização avançada)
- **[Swiper](https://swiperjs.com/)** (Carrosséis e slides)
- **[Axios](https://axios-http.com/)** (Requisições HTTP)
- **[Date-fns](https://date-fns.org/)** (Manipulação de datas)

## 🚀 Como Executar o Projeto Localmente

### Pré-requisitos
- Node.js (Recomendado v24.x)

### Passo a Passo

1. **Clone o repositório:**
   ```bash
   git clone <url-do-repositorio>
   cd <nome-do-repositorio>
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm start
   ```
   *O aplicativo estará disponível em `http://localhost:3000`.*

### Build para Produção
Para criar a versão otimizada do aplicativo:
```bash
npm run build
```
